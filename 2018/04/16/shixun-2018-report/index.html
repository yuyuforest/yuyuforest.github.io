不同于C/C++之处

Java和C++都是面向对象语言，在语法也很相似。但Java仍有些不同于C++的方面：

  • 一个java文件里最多有一个 public类，如果有，这个java文件的名字必须和这个类的名字一样。并且main函数要写在这个类里面，其签名统一为 public static void main(String[] args)
  • 不像C++可以把类内的函数的函数体写在类外面，java类的方法的方法体都必须写在类内。
  • java的接口机制是比C++的抽象类更抽象的东西，可以规定某些通用的方法和常量。一个类通过implements某个接口来实现这个接口，重写接口内的方法。
  • java有垃圾收集器的机制，可以自动监控对象并回收内存，这对于程序员来说是比C++更为便利的。（但并不意味着使用java就不需要手动回收内存了，这需要考虑程序的性能。）
  • java可以在类内再定义内部类，这个内部类可以使用它的外部类包含的方法和变量。

Vi/Vim自学报告

vi/vim是linux内的一款文本编辑器，可以直接在终端里配合按键编辑文本。

主要有两种模式：插入模式（insert mode）和命令行模式（command mode）。

插入模式可以对文本进行插入操作，命令行模式则对文本进行除插入以外的所有操作（如移动光标、删除文字、复制粘贴等等）。

除此以外还有一个底行模式（last line mode），可以进行保存或退出操作，将在下面对命令行的介绍中提到。

一般来说，在终端里用vi/vim打开文本时，默认处于命令行模式。

command mode 命令行模式

通过esc从insert mode切换到command mode。

常用操作如下：

移动光标

  • 使用键盘光标进行移动。
  • 使用字母键进行移动：h左移一格，而j、k、l对应下、上、右。
  • 0 : 移到文章开头
  • G : 移到文章末尾
  • $ : 移动到光标所在行的“行尾”
  • ^ : 行首
  • w : 跳到下个单词的开头
  • e : 跳到下个单词的末尾
  • b : 回到上个字的开头

删除文字

  • x : 删除光标所在位置的后面一个字符
  • X : 删除光标所在位置的前面一个字符
  • dd : 删除光标所在行

退出vi和保存文件

按一下「 : 」冒号键，进入Last line mode（底行模式），之后按以下某个键设置保存和退出：

  • w
  • w filename
  • wq
  • q!

撤销操作

  • u

insert mode 插入模式

在command mode下，如果要进入insert mode，有几种不同方法，每个方法的进入效果都不一样：

  • i : 进入插入模式后，从光标当前位置开始输入文字
  • a : 从光标当前位置的下一个位置开始输入文字
  • o : 插入新的一行，从行首开始输入文字

Ant自学报告

Ant可以实现项目的自动构建和部署等功能，简化我们对Java项目的编译运行过程。

构建文件

Ant 的构建文件是基于 XML 编写的，默认名称为 build.xml。

ant命令默认寻找build.xml，如果要寻找xxx.xml，有三种方式：

project元素

project 元素是 Ant 构建文件的根元素，Ant构建文件至少应该包含一个project 元素。

它的常用属性有：

  • name属性：用于指定project元素的名称
  • default属性：指定project默认执行时所执行的target名称。可不填写。
  • basedir属性：
    • 指定基路径的位置。该属性没有指定时，使用 Ant 的构件文件的附目录作为基准目录。

target元素

project 元素下，可包含多个 target 元素。它是Ant的基本执行单元，可以包含一个或多个要执行的具体的任务。

多个 target之间可以存在依赖关系。

  • name属性：target 元素的名称，这个属性在一个 project 元素中是唯一的

  • depends属性：描述 target 之间的依赖关系，若与多个 target 存在依赖关系时，需要以“,”间隔。Ant 会依照 depends 属性中 target 出现的顺序依次执行每个target。被依赖的 target 会先执行。

  • if属性：验证指定的属性是否存在，若不存在，所在 target 将不会被执行。

  • unless属性：功能与 if 属性的功能正好相反，它也用于验证指定的属性是否存在，若不存在，所在 target 将会被执行。

  • description属性：关于 target 功能的简短描述和说明。

property元素

可以设定project的参数或属性。也可在 Ant 之外设定。
