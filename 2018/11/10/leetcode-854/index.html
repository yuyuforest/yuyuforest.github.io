K-Similar Strings

Strings A and B are K-similar (for some non-negative integer K) if we can swap the positions of two letters in A exactly K times so that the resulting string equals B.

Given two anagrams A and B, return the smallest K for which A and B are K-similar.

暴力动态规划

根据网站所给的解答，还可以采用暴力动态规划来做这道题。

在进行规划之前，将字符看作节点，源字符串和目的字符串之间的转移关系是初始图，用一个长度为6*6的数组来表示这个图。

因为这个图中的交换次数等于 $∑(CL_i-1)$ ，其中 $CL_i$ 表示图中某个环 $C_i$ 的长度，

所以，这个图中的交换次数也等于 $(Number \ \ of \ \ nonself \ \ edges)−(Number \ \ of \ \ cycles)$ 。

如果要使交换次数最小，就令环数 $Number\ \ of\ \ cycles$ 最大。

然后，可以得出这样的关系：$numCycles(G)=1+numCycles(G-C)$ ，其中 $G-C$ 指从图中去除某个环 $C$ 后的新图。

接下来，暴力进行动态规划：求出所有环 $C_i$ ，有 $numCycles(G)=1+max{numCycles(G-C_1),numCycles(G-C_2), \cdots numCycles(G-C_n)}$ ，其中，参与运算的 $C_k$ 应该是能够被 $G$ 减去的环。

对 $G-C_k$ 继续进行如上的过程，这样递归下去，得到最终答案。
