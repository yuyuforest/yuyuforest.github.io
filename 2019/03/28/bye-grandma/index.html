怀逝

发表于 2019-03-28 | 更新于 2019-03-29 | 分类于 生活

纪念外婆。

前篇

我寒假回到家后，才知道外婆重病住院了，进了ICU。所幸，外婆有五个子女，都很孝顺。每周有三天上午，ICU有半小时开放探视，我每次去探视时都能看到好几个亲戚。

本来每年春节时，我们外婆的儿女子孙们都会聚会，但是今年因为外婆的病，大家都没心思聚在一起吃饭了。大年三十的晚上，大家几乎都聚在ICU门前，舅舅和表哥托了关系让我们轮流进去看一下外婆。我第一次在医院走廊看烟花。我第一次看到这样的“团圆”。

在变故面前，或许会有一些平时没有看到的东西浮现出来。我第一次去探视昏迷的外婆时，妈妈忙而无法前来，探视完后，她一直问我外婆有没有什么反应。我说外婆手指动了一下，她又追问是怎么动的，是怎么动的。我张口结舌，想，动了就是动了，还能怎么动啊？但她就是很在意这些细节，不停地问着。问完后，她就眼眶红红地走开了。年前大扫除时，她翻出一个旧手机，是外婆曾经学着使用过的；一张纸条，上面是外婆歪歪扭扭地写着妈妈的电话号码的笔迹。她还翻出了外婆塞到不知哪里的一叠钱，她又哭了。她已经失去了父亲，不知道什么时候会失去母亲。这可能是我一生中，第一次看到妈妈这么脆弱的样子。

有相熟的人怀上了第二胎。我说我将来可能不想生二胎，二胎养不起啊。妈妈说，子女多有子女多的好处，如果外婆只有独子，外婆昏倒时孩子不得吓死啊。而外婆的医疗费已经达到了十几万，几个子女分担起来也轻松一些。

在我返校前，我最后一次去探视外婆。外婆从ICU转入到神经外二科的病房，大家终于可以随时随地去探望了。我看着外婆被从移动床搬到病房的床上，护士给她翻身。同病房的也是不能自理的人，我看到他的亲属给他擦洗。我想，得了不能自理的重病真的很惨啊，多多少少都会失去一部分尊严，即便自己的亲人不介意，我自己也会介意。但愿将来我能努力确保自己身体健康，不得重病，至少要能够自理，不给亲人添麻烦。

希望外婆可以尽快好起来，可以少受点病痛的折磨，可以继续抱她的小孙女、小曾孙们。希望明年春节，外婆和我们一起喝早茶。

中篇

妈妈说：“千般不舍，万般无奈，几乎每天跑两次，只希望看到她稍睁开眼看我一下，尽管是无意识的……子欲养而亲不待，仿佛在耳边回响。”

后篇

前篇写于二月底。而昨天，2019.03.27，18:05，外婆永远地离开了我们。

转眼万事都成空。

心酸的是，我最后一次见到没有倒下的外婆，是在2018年的八九月。

遗憾的是，清明节的高铁票售空了，我也有课程、招聘要忙，无法回去祭拜外婆了。

想想，外婆80多岁了，也算是高寿了；生前看到了她的曾孙，经历了四世同堂，子孙孝顺，也算是无憾罢？不管怎样，至少她不用再在医院遭罪了。但愿她走的那时候，她没有痛苦。

家族里大概要有新的小生命了，人生或许就是不断地迎来送往吧。

想起我最喜欢的《前赤壁赋》：“……客亦知夫水与月乎？逝者如斯，而未尝往也；盈虚者如彼，而卒莫消长也。盖将自其变者而观之，则天地曾不能以一瞬；自其不变者而观之，则物与我皆无尽也，而又何羡乎？……”
